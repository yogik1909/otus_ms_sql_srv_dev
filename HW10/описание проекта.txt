Проект ведение домашней бухгалтерии
Проект нацелен на оцифровку действий по ведению личного бюджета.
Проект нацелен систематизировать учет расходов и доходов пользователя. Помочь анализировать поступающую информацию о тратах для принятия взвешенных решению по планирование личного и семейного бюджета.
Детализация транзакций позволит произвести наиболее точную оценку статей расходов и доходов человека.
База для проекта включает в себя набор метаданных для фиксации сами с транзакций и описание пользователей.
Талица tran_lines
Описывает детализацию по транзакции, сумма и категория затрат.
Таблица transaction
Аккумулирует все связанных строк транзакции из таблицы tran_lines
Таблица accounts
Счета ведения баланса. Это может быть счет в банке, кошелек, конверт или сейф.
Таблица view_accounts
Описывает сущность представления счета. Для счета в банке может быть выпущено несколько карт. Сейф может быть логически разбит на несколько конвертов. Для счета всего создается его идентичное описание, плюс дополнительное.
Таблица account_balance
Ведение баланса счет. Грубо говоря сальдовая ведомость.
Таблица users
Описание пользователя, владельца счетов
Таблица family
Группировка пользователей в одну группу. Для ведения общего бюджета.
